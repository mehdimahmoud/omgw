# 1. Clone the project
>git clone https://github.com/mehdimahmoud/omgw.git

Once you did it. 
  
# 2. Build the project
## With gradle
on windows terminal: 
>gradlew build

on linux/unix shell: 
>./gradlew build

## With docker
on windows: 
>gradlew buildDocker

on linux: 
>./gradlew buildDocker

This task carries out in one task two important tasks: 
1. build of the project
2. Create the docker image of the project


#3. Deploy the project locally
##1. With gradle
